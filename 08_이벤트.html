이벤트(Event)

1. 이벤트 모델 종류

* 고전 이벤트 모델( 기본 이벤트 모델)

특정 요소객체를 가지고 와서 해당 요소의 이벤트 속성에 접근한 후
이벤트 핸들러(이벤트가 발생했을 때 실행되는 함수) 연결하는 방식
이벤트 제거 할 수도 있음


*인라인 이벤트 모델

요소 내부에 직접적으로 이벤트 속성 제시해서 실행할 내용을 정의하는 방식
따로 script 코드에 작성된 함수를 작성하는 방식이다.


*표준 이벤트 모델

w3c에서 공식적으로 지정한 이벤트 모델

2. 현재 이벤트가 발생한 해당 요소객체에 접근하는 방법

3. 태그별 기본적으로 가지고 있는 이벤트 제거

기본적으로 이벤트 있는 요소
- a 태그 : 클릭시 href에 제시되어있는 url로 요청하는 기본이벤트 보유
- form 내의 submit 버튼 : 클릭시 사용자가 입력한 값들을 action에 제시되어있는 url로 제출

태그내에 기본적으로 설정되어있는 이벤트가 실행 안되게끔 막고싶다 .
=> 이벤트 핸들러의 return 값을 false로 하면 기본이벤트 실행 안됨.
